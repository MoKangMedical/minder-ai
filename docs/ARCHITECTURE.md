## 系统总览

Minder AI红娘是一个基于微服务架构的智能婚恋匹配平台，采用前后端分离设计，后端使用Python FastAPI框架，集成6大AI引擎提供智能匹配服务。

### 核心设计原则

1. **高可用**: 99.9%服务可用性
2. **高性能**: API响应时间 < 200ms
3. **可扩展**: 支持水平扩展
4. **安全性**: 数据加密、访问控制
5. **可维护**: 模块化设计、完善的文档

### 引擎1: 声纹情感分析

```
输入: 用户语音样本（30秒自我介绍）
    │
    ▼
预处理
    │
    ├─→ 降噪处理
    ├─→ 语音分段
    └─→ 特征提取 (MFCC, Spectrogram)
    │
    ▼
模型推理
    │
    ├─→ 情感识别模型 (BERT-based)
    ├─→ 性别/年龄预测
    └─→ 声纹特征提取
    │
    ▼
输出
    │
    ├─→ 情感倾向: positive/neutral/negative
    ├─→ 情绪稳定性: 0-100
    ├─→ 沟通风格: direct/indirect/expressive
    └─→ 声纹向量: 128维特征向量
```

```
    │
    ├─→ 人脸检测 (MTCNN)

模型推理
    │

    ├─→ 性格推断: open/conscientious/extraverted/...
    └─→ 面部特征向量: 256维特征向量
```

### 核心表结构

#### users 表
```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    phone VARCHAR(20) UNIQUE,
    gender VARCHAR(10) NOT NULL,
    birth_date DATE NOT NULL,
    city VARCHAR(50),
    avatar_url VARCHAR(500),
    is_verified BOOLEAN DEFAULT FALSE,
    is_premium BOOLEAN DEFAULT FALSE,
    is_active BOOLEAN DEFAULT TRUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### JWT认证流程

```
1. 用户登录
   │
   ▼
2. 验证凭据
   │
   ▼
3. 生成JWT Token
   │
   ├─→ Access Token (1小时有效期)
   │   {
   │     "sub": "user_id",
   │     "exp": "过期时间",
   │     "type": "access"
   │   }
   │
   └─→ Refresh Token (30天有效期)
       {
         "sub": "user_id",
         "exp": "过期时间",
         "type": "refresh"
       }
   │
   ▼
4. 客户端存储Token
   │
   ▼
5. 请求携带Token
   Authorization: Bearer {access_token}
   │
   ▼
6. 服务端验证Token
   │
   ├─→ 验证签名
   ├─→ 验证有效期
   └─→ 提取用户信息
   │
   ▼
7. 处理请求
```
